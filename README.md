# 高校野球指導アドバイザーAI開発のための野球ノート分析（自然言語処理、感情分析、クラスタリング）

## 概要
日揮パラレルテクノロジーズ（JPT）体験インターン課題として作成した成果物です。
高校野球の『野球ノート』を自然語処理で解析し、選手の心理傾向の可視化、特徴量抽出、クラスタリングによるタイプ分類を行いました。
解析内容の詳細は、PDFレポートにまとめています。
[PDFレポートはこちら](PDF/2025年度%20日揮体験インターン.pdf)

## 使用技術
- Python/Jupyter Notebook  
- MeCab・fugashi（形態要素解析）
- TF-IDF（特徴量抽出）
- matplotlib/seaborn（可視化）
- K-means/PCA（クラスタリング）
- 感情分析（独自ルール + 自然言語処理）

## プロジェクト概要
1. 与えられた選手の性格をもとに野球ノートを生成  
2. ノートから選手の感情を分析  
3. 選手のタイプを分類し、どの監督またはコーチに指導してもらうか提案  
4. 今後の展望として、AI開発について提案  

PDFには、背景、目的、分析フロー、結果、今後の展望を詳細に掲載しています。
